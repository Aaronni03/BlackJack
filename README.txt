To start the game, compile all java files by double clicking "Compile_Run.bat". 
If you have already compiled all files, use "run.bat".

If you don't have the environment variable points to JavaFX library:
Use batch file "Compile_Run.bat" as a reference on how to import modules from JavaFX library. 
Must have JDK and JRE 9 installed on your computer.

Current version of the game only supports 1 to 3 players.

When you have depleted your balance, you must restart the game to continue playing.
